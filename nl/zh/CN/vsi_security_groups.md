# 安全组

通过 {{site.data.keyword.BluVirtServers}} 的安全组，可以制定一组 IP 过滤规则，用于定义如何处理与虚拟服务器实例的公共接口和专用接口的传入和传出流量。一个安全组会创建一种虚拟防火墙。
{:shortdesc}

有关安全组功能的更多信息，请参阅[安全组入门](/docs/infrastructure/security-groups/sg_index.html)。
