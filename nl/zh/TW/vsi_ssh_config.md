# 配置 SSH 金鑰
Secure Socket Shell (SSH) 通訊協定是伺服器上所安裝之作業系統的一部分。您有數個配置選項可用（例如，配置埠，以及限制存取權）。您也可以管理 SSH 金鑰。

如需 SSH 選項的相關資訊，請參閱[開始使用 SSH 金鑰](/docs/infrastructure/ssh-keys/index.html)。
