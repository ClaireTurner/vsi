# 配置 SSH 密钥
{: #configuring-ssh-keys}

安全套接字 Shell (SSH) 协议是服务器上安装的操作系统的一部分。您有若干配置选项可用，例如配置端口和限制访问。还可以管理 SSH 密钥。

有关 SSH 选项的更多信息，请参阅 [SSH 密钥入门](/docs/infrastructure/ssh-keys?topic=ssh-keys-getting-started-tutorial)。
