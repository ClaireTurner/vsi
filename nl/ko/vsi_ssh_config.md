# SSH 키 구성
SSH(Secure Socket Shell) 프로토콜은 서버에 설치된 운영 체제의 일부입니다. 포트 구성 및 액세스 제한과 같이 몇 개의 구성 옵션을 사용할 수 있습니다. SSH 키도 관리할 수 있습니다.

SSH 옵션에 대한 자세한 정보는 [SSH 키 시작하기](/docs/infrastructure/ssh-keys/index.html)를 참조하십시오.
